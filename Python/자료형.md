# 자료형

### 1. 숫자형

- 정수형
- 실수형
- 8진수, 16진수
- 사칙연산
- 제곱`**`, 나머지 `%`, 몫 `//`



### 2. 문자열

- 문자들의 집합
- 이스케이프 코드
- 연산 (더하기, 곱하기)
- 인덱싱과 슬라이싱
- 포맷팅



### 3. 리스트 (List)

- 변수들의 모음
- 인덱싱과 슬라이싱
- 연산 (더하기, 곱하기, 수정, 삭제)



### 4. 튜플 (Tuple)

​	:star:**변경할 수 없음**

- 인덱싱과 슬라이싱
- 연산 (더하기, 곱하기, 수정, 삭제)



### 5. 딕셔너리 (Dictionary)

​	`Key : Value` 한 쌍

- 추가, 삭제



### 6. 집합 (Set)

​	**중복이 불가능**

​	**순서가 없음**

- 교집합
- 합집합
- 차집합
