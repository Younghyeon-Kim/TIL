## Command

- `프로그램 + 명령 + what`

  ex) `conda install pandas`

  - 설치 : `install`

  - 제거 : `remove`

  - 찾기 : `search`

  - 생성 : `create`

  - 활성화 : `activate`

  - 비활성화 : `deactivate`

  - 가상환경 확인하기

    `conda env list` 

    **:star:현재 `activate` 되어있는 곳에 * 표시**

  - 가상환경 지우기

    `conda env remove -n 가상환경`

    ex) `conda env remove -n basic`
