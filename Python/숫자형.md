# 숫자형

### 종류

#### 1. 정수

​	ex) `123`, `-345`, `0`

#### 2. 실수

​	ex) `123.45`, `-1234.5`

- 지수표현 : `3.4e10`

#### 3. 8진수

​	ex) `0o34`, `0o25`

#### 4. 16진수

​	ex) `0x2A`, `0xFF`

---



### 연산자

#### 1. 사칙연산

- 더하기 : `+`
- 빼기 : `-`
- 곱하기 : `*`
- 나누기 : `/`

#### 2. 제곱연산

- 제곱 : `**`

#### 3. 나머지 연산

- 나머지 : `%`

#### 4. 몫 연산

- 몫 : `//`
